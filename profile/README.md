# Monster Gaming

**AI-powered game development platform.**

Monster Gaming is an AI platform that sits above the engine. Code generation, debugging, optimization, and asset pipelines — purpose-built for [Unreal Engine](https://monstergaming.ai), Unity, Godot, and bespoke engines.

## What We Build

- **[Monster-GPT](https://monstergaming.ai)** — flagship model that auto-routes your query to one of 30+ specialist agents (shader, animation, netcode, level design, QA, and more)
- **[OpenAI-compatible API](https://monstergaming.ai/quickstart)** — drop-in replacement for OpenAI SDKs with game-dev-tuned models
- **[Engine-aware code generation](https://monstergaming.ai)** — writes gameplay systems, shaders, networking, and UI that actually compile
